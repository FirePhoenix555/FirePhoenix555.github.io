About me, and this site

Hello!

I'm fire (he/him). I'm a freshman (first-year) at a college in California, though I'm originally from the Midwest.

On this site, I host my interests: math, linguistics, Latin, and of course programming. I also host my projects, which you can check out on the Projects page.

I'm a Linux (Ubuntu) and Firefox user. I write code with VS Code mainly, and IntelliJ IDEA or Processing for Java specifically. I'm also familiar with LaTeX.

About my projects

  • Chatapp [WIP]: a website and terminal interface for communication between friends.
  • Fourier series: a program to recreate user drawings with rotating circles.
  • On Thin Ice: a game about a lost baby polar bear looking for its mother. A collaboration with GoldenStack.
  • DMP bot: a Discord bot to post daily math problems.
  • Duodinguistics [WIP]: a game about deciphering languages. A collaboration with CabbageOfCato.
  • Certamen buzzer [WIP]: a button to register a buzz in virtual Certamen.
  • CertamenbotV2 [WIP]: a recreation of the old Discord Certamenbot.
  • Night [WIP]: a game about anxiety.

You can also ask me about my conlanging projects: Furasen, Ponamukkā, Tisīcan, Laōyāji, Anen, and more.

  • Math: I have taken math classes through Calculus III, as well as Differential Equations and Linear Algebra.
  • Programming: I have the most experience in JavaScript and Java, but I've also dabbled in Python, C++, bash, and a few others. I've been coding for almost 10 years now!
  • Language: I'm a native English speaker, but I've studied Latin (and Spanish) in school and looked at a few other languages casually too.
  • Certamen: I'm an alum now, unfortunately, but I played language for 4 years while I was still a JCLer. Grammar>>> :D
